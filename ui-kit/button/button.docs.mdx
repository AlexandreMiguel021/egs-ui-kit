Componente de botão.

Ao passar href para o componente Button, ele será renderizado como um componente "anchor" no html.

```js
<Button href="/page">Hello world!</Button>
```

No Next.js para ter o benifício do roteamento você pode envolver o componente "Button" com o componente "Link".

```js
<Link href="/page" passHref>
  <Button>Hello world!</Button>
</Link>
```

---

> ❕ **info:** Em dispositivos menores que 768px, o botão tem o tamanho padrão _sm_.

## Uso do componente

```js
<Button>Hello world!</Button>
```

```js live
<ThemeProvider theme={theme}>
  <Button>Hello world!</Button>
</ThemeProvider>
```

```js live
<ThemeProvider theme={theme}>

</ThemeProvider>
```

```js live

```


